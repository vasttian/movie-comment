# movie-comment

## 技术栈（后续添加）

+ Node.JS
+ Express
+ MongoDB
+ AngularJS
+ Bootstrap
+ BootstrapValidator
+ ECharts
+ Moment
+ Mongoose
+ Session
+ Underscore


## use
- npm install
- bower install

- 登录注册（邀请码）

- 电影的CRUD
- 电影分类的CRUD
- 个人资料查看
- 个人资料修改
- 密保设置、找回密码
- 电影预告片等详情查看
- 电影评论以及相互评论
- 电影搜索、类别查看
- 自动同步豆瓣数据
